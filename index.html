🔧 Firebase 設定

首次使用需要設定 Firebase 專案資訊

📋 設定步驟

  1. 前往 Firebase Console
  2. 建立新專案或選擇現有專案
  3. 啟用 Authentication → Google 登入方式
  4. 建立 Realtime Database
  5. 前往「專案設定」→「一般」→「您的應用程式」
  6. 選擇「網頁應用程式」，複製設定資訊並填入下方

⚠️ 提示：設定完成後會自動重新載入頁面。這些設定會儲存在您的瀏覽器中。
